# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档及答辩PPT
- 参考导入调试视频、讲解视频、开发工具安装包
- 库存预警功能
- Echarts可视化统计分析
- 员工、客户信息管理
- 货物分类管理
- 入库信息管理
- 订单信息管理
- 出库信息管理
- 物流信息管理
- 过期处理
- 盘点信息管理
- 反馈建议管理
- 运营分析
- 菜单列表管理
- 个人中心

## 适用场景
- 物流仓储企业
- 物流配送中心
- 供应链管理公司

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
